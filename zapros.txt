Задание № 1.
Решение:
SELECT "inDelivery", 
count(login) 
FROM "Couriers","Orders" 
WHERE "inDelivery" = true 
GROUP BY "inDelivery";


Задание № 2.
Решение:
SELECT track, 
CASE  
WHEN finished = true THEN 2 
WHEN canсelled = true THEN -1 
WHEN "inDelivery" = true THEN 1 
ELSE 0 
END  AS status 
FROM "Orders";
